## 表达式和语句

* 表达式：一个和多个运算数加上一个或多个的操作符所组成的序列

* 语句：

* Python中常用的表达式操作符
    低优先级：+,- 
    高优先级：*,/,//,%  
    
  逻辑运算：  
    or,and,not  
    
  成员运算：  
    in，not in  

  实例测试：
    is，not is
  
  比较运算：
    <,>,<=,>=,<>,!=,,==
    
  位运算：
    |，&，^,<<,>>
    
  一元运算：
    -，+，~
  
  幂运算：
    **
    
  索引和分片：
    s[i],s[i:j],s[i:j:stride]
    
  调用:
    x(...)
  
  点运算：
    x.attribution
  
  元组：s()
  序列：s[]
  字典：s{}
  集合：set(),frozenset()
  
  三元选择表达式：
    x if y else z
    
  匿名函数:lambda args:expression
  
  生成器函数发送协议:yield x
