### Python命名惯例

* 以单一下划线开头的变量名（标识符）\_不会被 from module import * 语句导入
* from module import 导入形式会覆盖现有文件中的变量名，而直接使用import不会覆盖，

  如果用import导入的模块中有和现有文件中变量名，函数名重名的，要使用
  
  模块名.变量名  
  
  例如：
        import module
  
        module.methon()
