## Else在for循环中的使用说明

```
>>> for n in range(2, 10):
...   for x in range(2, n):
...     if n % x == 0:
...       print n, 'equals', x, '*', n/x
...     break
...   else:
...   # loop fell through without finding a factor
...     print n, 'is a prime number'
...
2 is a prime number
3 is a prime number
4 equals 2 * 2
5 is a prime number
6 equals 2 * 3
7 is a prime number
8 equals 2 * 4
9 equals 3 * 3
```
注意：这里的else是跟在for循环后面的，不是if后面的else
这里的else后的代码是等到for x in range(2, n) 的循环条件变为假后才开始执行
